# ExpiryTracker 🥫

A smart Android app that helps users track product expiry dates 
and get notified before items expire — keeping your home 
waste-free and organized.

## Features ✨
- 📸 **Smart Product Entry** — No manual typing needed!
  Uses OCR + Image Enhancement + AI to automatically
  read product details from photos
- ⏰ **Expiry Tracking** — Track all your products
  and their expiry dates in one place
- 🔔 **Expiry Notifications** — Get notified before
  your products expire
- 🛒 **Buy Again** — Easily reorder products that
  are about to expire or have expired
- 🍳 **Recipe Suggestions** — Get recipe ideas based
  on products you have
- ♻️ **Disposal Tips** — Get eco-friendly disposal
  tips for expired products
- 🔥 **Streak Tracker** — Stay consistent with
  tracking your products

## Tech Stack 🛠️
- Kotlin
- Android Studio
- Room Database
- OCR (Text Recognition)
- Image Enhancement
- Groq AI API
- OpenRouter AI API (Gemma 3)

## Setup ⚙️
1. Clone the repository
2. Open in Android Studio
3. Create a `local.properties` file in the root folder
4. Add your API keys:
5. Sync Gradle and run the app

## Important 🔐
Never hardcode API keys in your source code.
Always use local.properties to store sensitive keys.
